Проверка фильтров Django

Данные Фильтр Результат
{{ user_name }} upper (в верхний регистр) {{ user_name|upper }}
{{ user_name }} capfirst (первая буква заглавная) {{ user_name|capfirst }}
{{ long_text }} truncatewords:5 (обрезать до 5 слов) {{ long_text|truncatewords:5 }}
{{ today }} date:"d E Y" (формат даты) {{ today|date:"d E Y" }}
"{{ empty_var }}" default:"Ничего не найдено" {{ empty_var|default:"Ничего не найдено" }}
{{ item_list }} length (количество элементов) {{ item_list|length }}
